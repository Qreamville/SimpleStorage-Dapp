# FundMe-Dapp
A simple storage decentralise application. Used to store favorite number. 


### Built With
* [Solidity](https://docs.soliditylang.org/en/v0.8.11/)
* [Hardhat](https://hardhat.org/)
* [TailwindCss](https://tailwindcss.com/)
